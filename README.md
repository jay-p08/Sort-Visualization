# 📊 Sorting Algorithm Visualizer

정렬 알고리즘을 **시각적으로 이해하기 어려워하는 학생들**을 위해 만든  
정렬 알고리즘 시각화 프로그램입니다.

각 정렬 알고리즘이 **어떤 순서로 비교하고, 언제 교환이 일어나는지**를  
화면을 통해 직관적으로 확인할 수 있도록 구현했습니다.

---

## 🎯 제작 목적

정렬 알고리즘은 이론으로 보면 이해한 것 같지만  
코드나 문제에 적용하려고 하면 헷갈리는 경우가 많습니다.

이 프로젝트의 목적은 다음과 같습니다.

- 정렬 알고리즘의 **동작 과정을 시각적으로 이해**
- 알고리즘 입문자에게 친화적인 학습 도구 제공
- 코드 암기가 아닌 **원리 중심 학습** 지원

---

## 🧠 구현된 정렬 알고리즘

- Bubble Sort
- Selection Sort
- Insertion Sort
- Heap Sort
- Merge Sort
- Quick Sort

각 알고리즘은 다음 요소를 시각화합니다.

- 현재 비교 중인 값
- 교환(Swap) 발생 여부
- 정렬 진행 단계

---

## 🖥️ 주요 기능

- 랜덤 배열 생성
- 정렬 과정 실시간 시각화
- 알고리즘별 동작 비교
- 단계별 애니메이션 표시
